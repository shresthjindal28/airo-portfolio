# Aevomed Portfolio (Next.js)

Company portfolio site for **Aevomed**, built with Next.js App Router.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3002](http://localhost:3002).

## Build

```bash
npm run build
npm start
```

## Deploy

Deploy to **Vercel** — import the repo, framework preset **Next.js**, no extra config needed.

## Structure

```
src/
  app/           layout, page, globals.css
  components/    header + page sections
  lib/content.ts site copy & links
```

Edit `src/lib/content.ts` to update products, features, and links.
